# Servidor frutas

## Iniciar el proyecto

```bash
npm init -y
```

Esto genera un `package.json` con la configuracion por defecto

## Instalar librerias
```bash
npm i <paquete>
npm i install <paquete>
npm install express
```

## Importar las libreas que hemos usado

```javascript
const val = require("paquete")
const express = require("express") //importarla en una variable
```

## Ejecutar js
```bash 
node index.js
node server.js
```
